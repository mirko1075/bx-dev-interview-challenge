S3 Presigned URLs Demo

1. Authentication

First, you need to authenticate and get a JWT token:

No token set

2. Upload File with Presigned URL

Select a file and click upload

3. Generate Download URL

Enter an S3 key and click generate

4. API Usage Examples

Get Presigned Upload URL:

POST /files/presigned-upload
Authorization: Bearer <token>
Content-Type: application/json

{
  "fileName": "example.jpg",
  "fileType": "image/jpeg",
  "expiresIn": 3600
}

Upload directly to S3:

PUT <presigned-upload-url>
Content-Type: image/jpeg

<file-data>

Get Presigned Download URL:

POST /files/presigned-download
Authorization: Bearer <token>
Content-Type: application/json

{
  "key": "uploads/user-123/uuid-example.jpg",
  "expiresIn": 3600
}
